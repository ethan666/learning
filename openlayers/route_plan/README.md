# route_plan

路线规划，输入起点、终点显示路线规划方案

1. 界面采用哪种 layers
2. 输入框绘制， 界面绘制
3. 导航信息展示， 每一步怎么走，前方直行，600m 右
4. 多个路线之间切换
   绘制线路
